# Log

Module contain functions for logging

---

# Methods

## Info

```lua
function Log.Info(where, msg, ...)
end
```

Log info message.

### Arguments

Argument | Type | Description
---------|------|-------------------------
`where`  | str  | Where is message logged.
`msg`    | str  | Message.
`...`    | str  | Message args.
