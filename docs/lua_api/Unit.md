### spawn( world, name ) : unit

#### Returns
* Spawned unit

#### Arguments
* **world** `world` - World
* **string** `name` - Unit resource name

### destroy( world, unit )

Destroy unit

#### Arguments
* **world** `world` - World
* **unit** `unit` - Unit
