# Log

Module contain functions for logging

## Methods

### Info(where, msg, ...)

Log info message.

Argument | Type | Description
---------|------|-------------------------
`where`  | str  | Where is message logged.
`msg`    | str  | Message.
`...`    | str  | Message args.
